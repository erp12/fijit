# Fijit

A Clojure library for Scala interop.

# Motivation

Clojure and Scala are both:
- **F**unctional.
- **I**mmutable-first.
- **J**vm hosted.

**Q:** Why `fijit`? **A:** Easier to call Scala projects. Spark and Akka.  

# Usage

- Pin to git commit.
- Provide your own scala version.
